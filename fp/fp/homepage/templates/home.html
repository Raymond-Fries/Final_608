About

This project is about using the Alpaca.Markets trading api to create a algorithmic trading system. That will buy and sell stocks based off of historical data.

Data Analysis

In this project the data analysis is pretty vanilla, as I am concerned about the legal ramifications of someone trying to use this system to trade real money.

With that said, I have created database tables that stores the mean of each minute. I have also derived the mean for each week days minute mean. Both tables have the fields of the minutes mean and the standardized mean for easy conversion for pricing. Although I am not using a pricing mechanism in this system.

Data Source

To retrieve the data, the application is connected to the alpaca.markets streaming service. The data consists of an agregate(open, high, low, close) of the minute by minute trades for Apple and Advanced Micro Devices. This is streamed from between 9:30 am to 4:00 pm eastern standard time.

Visualizations

For the actual visualizations, the page is made up of several different pieces that we have used throughout the class. There is a dropdown list for selecting which companies chart to view.

There are also three tables. One table that displays a cash balance, buying power and profits for the day.

Another table that shows which positions are held in which companies. The third and final table shows all the transactions made during the day. There is also a text box with this table that shows how many transactions have occurred during the day.

In the charts there are two simple moving averages. A 30 minute simple moving average and a 60 minute simple moving average. Both of which provides a smoothed visualization of a shorter more responsive trend line(3O ma) and a longer less responsive trend line(60 ma)

Visualizations Color Scheme

We are all unique and have our own characteristics. For instance, I have light colored eyes. Which are sensitive to bright lights. When I walk outside during the day I will squint constantly as the light bothers them.

It's the same thing when I sit at a computer all day staring at a screen that is emitting light. It strains my eyes.

So the color scheme that I chose to use mainly has this issue in mind. Using a dark blue background that is easy on my eyes and contrasting it with white, grey, yellow, green and red.

Why is this Data Important?

This data is important as it allows me to control my financial destiny by creating a trading algorithm that will make decisions based on the data for me.

However, there are many variables that go into the movements of stock prices. As is, this system is does not account for 99% of the other variables.
